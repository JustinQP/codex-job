## 三、前端页面调整

### 1. App.tsx

调整 Tab 定义：

旧：

home / tasks / app / settings

新：

app / projects / runs / settings

页面标题：

* app：会话
* projects：项目
* runs：运行
* settings：我的

页面副标题：

* app：连续对话控制
* projects：当前工作目录与项目配置
* runs：后台运行记录
* settings：连接、诊断与维护

默认：

useLocalStorage(UI_STATE_KEYS.activeTab, "app")

如果 localStorage 里仍是旧值 home/tasks，需要做兼容映射：

* home -> app
* tasks -> runs
* app -> app
* settings -> settings

### 2. BottomNav

底部导航改成：

会话 / 项目 / 运行 / 我的

### 3. 删除或下线 HomePage

HomePage 不再作为一级页面。

可以选择：

方案 A：删除 HomePage 组件和入口。

方案 B：暂时保留文件但不再从 App.tsx 引用。

推荐 A，但如果改动风险大，先 B。

首页里的状态摘要能力可以迁移到：

* 项目页：显示当前项目、最近会话
* 我的页：显示 Backend / Runner / Bridge 状态
* 运行页：显示运行中/失败/最近完成

### 4. 新增 ProjectsPage

新增：

frontend/src/components/projects/ProjectsPage.tsx

功能目标：

1. 显示项目列表。
2. 支持选择当前项目。
3. 当前项目保存到 localStorage：
   mobile.currentProjectId
4. 会话页创建新会话时默认使用当前项目。
5. 如果没有 currentProjectId，默认使用第一个 enabled project。
6. 项目页不要做复杂项目管理，只做轻量选择与查看。

页面结构建议：

* 当前项目卡片

  * 项目名称
  * 路径 label
  * 默认 runner / model / sandbox
* 项目列表

  * 点击项目设为当前项目
* 最近会话

  * 点击进入会话页并选中该会话

使用现有 API：

GET /projects
GET /app-threads?limit=5

不要新增后端 API。

### 5. 会话页增强 current project 支持

SessionPage 创建会话时使用当前项目：

1. 读取 localStorage mobile.currentProjectId。
2. ThreadSwitcherSheet 的“快速新建”默认项目使用 currentProjectId。
3. 创建成功后继续选中新会话。
4. 如果 currentProjectId 不存在，使用 projects[0]。
5. 会话 header 或切换 sheet 中可轻量显示当前项目名，但不要让会话页变复杂。

会话页仍然是主页面，不要把项目选择做成大表单。

### 6. TasksPage 改名为 RunsPage

把“任务”概念降级为“运行”。

建议：

* frontend/src/components/tasks/TasksPage.tsx 可以迁移/重命名为：
  frontend/src/components/runs/RunsPage.tsx

* TaskCard 可以迁移/重命名为：
  RunCard.tsx

如果一次性重命名风险大，可以先保留文件名，但 UI 文案必须改为“运行”。

UI 文案调整：

* 任务中心 -> 运行记录
* 新建任务 -> 删除主入口
* 任务状态筛选 -> 运行状态
* 没有匹配任务 -> 没有匹配运行记录
* 查看任务 -> 查看运行
* 重跑 / 取消 放到更多或详情中，别直接铺在列表上

RunsPage 只负责：

1. 运行中
2. 失败
3. 最近完成
4. 查看日志 / result / diff
5. 必要时重跑 / 取消

不要在 RunsPage 主界面显示“新建任务”按钮。

如仍需保留高级创建能力，放到：

我的 -> 维护操作 -> 高级创建任务

本阶段可以先不做高级创建入口。

### 7. TaskCard / RunCard 轻量化

运行列表项主操作：

* 点击整行：打开详情

低频操作：

* 重跑
* 取消
* 查看日志
* 查看 result
* 查看 diff

放进详情 Sheet 或更多操作中。

不要每条记录直接显示三个大按钮。
