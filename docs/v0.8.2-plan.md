# 二、任务 1：新增 App Server 主线 smoke 脚本

新增文件：

```text id="bnw4bt"
scripts/smoke_app_server_flow.py
```

目标：

通过主后端 API 完成一次 App Server 会话端到端验收。

脚本只访问主后端 `http://127.0.0.1:8000`，不要直接访问 `8766` Bridge。

## 支持参数

```text id="4vbt9n"
--base-url 默认 http://127.0.0.1:8000
--token 可选，默认读取 API_TOKEN
--project-id 可选
--project-name 默认 smoke-app-server-project
--project-path 默认当前工作目录
--title 默认 Smoke App Thread
--message 默认 请只回复 app-thread-smoke-ok，不要修改文件。
```

## 行为

1. 调用：

```text id="t1f3dz"
GET /health
GET /app-server-bridge/health
```

2. 如果未传 `--project-id`：

```text id="iedp8t"
POST /projects
```

创建一个项目。

3. 调用：

```text id="tx3kgo"
POST /app-threads
```

创建 App Thread。

4. 调用：

```text id="w5f2y7"
POST /app-threads/{id}/turns
```

发送 message。

5. 调用：

```text id="ope02v"
GET /app-threads/{id}/final
GET /app-threads/{id}/turns
GET /app-threads/{id}/events
```

6. 最后调用：

```text id="j13p0k"
DELETE /app-threads/{id}
```

关闭 App Thread。

## 输出

控制台输出清晰 JSON 摘要：

```text id="6tf7tb"
backend_health
bridge_health
project_id
app_thread_id
turn_id
assistant_final
events_summary
closed_status
pass: true/false
```

## 判定通过

如果 `assistant_final` 包含：

```text id="esg3iu"
app-thread-smoke-ok
```

则 `pass=true`。

如果失败：

1. 输出失败步骤。
2. 输出 HTTP status。
3. 输出响应 body。
4. 退出码为 1。

成功退出码为 0。

# 四、任务 3：后端 API 小增强

修改文件：

```text id="sijej5"
backend/main.py
backend/services/app_thread_service.py
backend/schemas.py
```

## 1. AppThreadRead 增加 latest_assistant_final

在 `AppThreadRead` 增加字段：

```text id="fcfw3t"
latest_assistant_final: Optional[str] = None
```

`to_app_thread_read()` 返回该 thread 最新 SUCCESS turn 的 `assistant_final`。

用途：

手机列表可以直接看到最近回复摘要，不用每次点进去再拉 turns。

注意：

1. 不新增数据库字段。
2. 通过查询 `AppTurn` 获取最新 SUCCESS turn。
3. 如果没有成功 turn，返回 None。

## 2. AppThreadRead 增加 turn_count

在 `AppThreadRead` 增加字段：

```text id="ep27ic"
turn_count: int = 0
```

`to_app_thread_read()` 返回该 thread 的 turn 数量。

注意：

1. 不新增数据库字段。
2. 通过查询 `AppTurn` count 或简单查询列表长度实现。
3. 保持性能简单即可，当前 MVP 规模不大。

## 3. 修改调用点

当前 `to_app_thread_read(app_thread)` 只有一个参数。

可以改成：

```python id="g4k1o0"
to_app_thread_read(session, app_thread)
```

或者新增：

```python id="1hzie9"
to_app_thread_read_with_stats(session, app_thread)
```

要求：

1. 所有 API 返回 AppThreadRead 的地方都带上 `latest_assistant_final` 和 `turn_count`。
2. 不影响已有字段。
3. 测试要同步更新。
